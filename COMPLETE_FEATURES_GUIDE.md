## 🎵 音乐播放器系统

### 路由访问
- **音乐页面**: `/music`

### 核心功能
- ✅ **完整音乐库管理**
  - 文件夹结构管理（每首歌一个文件夹）
  - 支持音频、封面、歌词、元数据、时间戳
  - 搜索、筛选、排序功能
  - 网格/列表视图切换
  - 下载、收藏功能
- ✅ **奏词同步显示**
  - LRC格式歌词文件
  - 实时滚动同步
  - 时间戳精确匹配
- ✅ **自动化工具**
  - 一键生成音乐元数据
  - 自动创建文件夹结构
  - 批量处理音乐文件

### 文件结构
```
public/music/music list/
└── [歌曲文件夹]/
    ├── audio.mp3          # 音频文件
    ├── cover.jpg          # 封面图片
    ├── lyrics.lrc         # 歌词文件
    ├── metadata.json      # 元数据
    └── timestamps.json    # 时间戳
```

### 🎤 语音识别歌词生成（新功能）
- **功能**: 使用Web Speech API自动识别音频中的歌词
- **使用方法**:
  1. 播放任意音乐
  2. 点击播放控制栏的麦克风图标 🎤
  3. 点击"开始识别"
  4. 允许麦克风权限
  5. 系统自动播放并识别歌词
  6. 生成LRC格式歌词文件
- **支持浏览器**: Chrome、Edge、Safari
- **特点**: 实时识别、精确时间戳、一键保存

### 使用方法
1. 访问 `/music` 页面
2. 点击音乐卡片播放
3. 使用搜索和筛选功能
4. 点击麦克风图标进行语音识别
5. 下载或保存生成的歌词

## 📝 笔记系统

### 路由访问
- **笔记首页**: `/notes`
- **创建笔记**: `/notes/editor`
- **查看笔记**: `/notes/view/:id`

### 功能特性
- ✅ **三大笔记分类**
  - **生活**（粉色主题）- 日常记录、心情随笔
  - **随笔**（紫色主题）- 思考感悟、读书笔记
  - **算法**（绿色主题）- 编程题目、算法学习
- ✅ **美观的界面设计**
  - 卡片式首页布局
  - 统计卡片展示
  - 彩色分类标签
  - 响应式设计
- ✅ **完整的编辑功能**
  - Markdown编辑器
  - 实时预览
  - 标签管理
  - 算法特殊字段（题号、难度、复杂度）
- ✅ **内容缩放功能**
  - 50%-200%缩放范围
  - 只影响内容，不影响页面布局
  - 平滑过渡动画
  - 查看和编辑模式都支持

### 🎨 图形编辑器（升级功能）
- **功能**: 自由图形绘制，类似draw.io
- **使用方法**:
  1. 创建或编辑笔记
  2. 点击顶部操作栏的"📊 添加图表"按钮
  3. 在图形编辑器中选择形状和颜色
  4. 拖拽图形到画布上
  5. 双击图形编辑文字
  6. 保存图表嵌入到笔记中
- **支持的图形**:
  - 矩形、圆形、菱形、三角形、星形、文本
- **颜色选择**: 7种预设颜色可选
- **特点**: 拖拽添加、双击编辑、自由组合

### 🔍 缩放功能（优化）
- **功能**: 使用浏览器原生缩放
- **使用方法**:
  - **放大**: `Ctrl + +` 或 `Ctrl + 滚轮向上`
  - **缩小**: `Ctrl + -` 或 `Ctrl + 滚轮向下`
  - **重置**: `Ctrl + 0`
- **优势**: 更精确、更流畅、自动记忆

### 标签系统
- **支持标签**: 生活、随笔、算法、LeetCode（向后兼容）
- **标签更新**: 已更新支持"算法"标签，同时兼容旧的"LeetCode"标签

### 使用方法
1. 访问 `/notes` 查看笔记首页
2. 点击"写笔记"创建新笔记
3. 选择分类和添加标签
4. 使用Markdown编写内容
5. 添加图形图表（可选）
6. 保存笔记

## 🔧 故障排除

### 常见问题

#### 音乐播放问题
- **问题**: 音乐无法播放
- **解决**: 检查文件路径和格式，确保音频文件存在

#### 语音识别问题
- **问题**: 语音识别不工作
- **解决**: 使用Chrome浏览器，允许麦克风权限

#### 图形编辑问题
- **问题**: 找不到添加图表按钮
- **解决**: 确保在编辑页面，按钮在顶部操作栏

#### 笔记显示问题
- **问题**: 笔记内容不显示
- **解决**: 检查localStorage数据，刷新页面

### 浏览器兼容性
- **推荐**: Chrome、Edge、Safari
- **语音识别**: 需要支持Web Speech API
- **图形编辑**: 需要支持现代CSS和SVG
