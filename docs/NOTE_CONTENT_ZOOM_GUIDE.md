# 📝 笔记内容缩放功能指南

## 🎯 功能概述

笔记页面现在支持**内容缩放**功能，允许用户独立调整笔记内容的大小，而不影响整个页面的布局。

---

## ✨ 功能特点

### 🔍 内容缩放 vs 页面缩放
- **内容缩放**: 只影响笔记内容的显示大小
- **页面缩放**: 影响整个页面的所有元素

### 🎛️ 缩放控制
- **缩放范围**: 50% - 200%
- **缩放步长**: 每次调整10%
- **默认缩放**: 100%
- **平滑过渡**: 缩放变化有0.2秒的过渡动画

---

## 📍 功能位置

### 1. **笔记查看页面** (`/notes/:id`)
- 位置: 操作栏右侧，编辑和删除按钮之前
- 显示: 缩放控制按钮组

### 2. **笔记编辑页面** (`/notes/editor`)
- 位置: 仅在预览模式下显示
- 显示: 缩放控制按钮组

---

## 🎮 使用方法

### 缩放控制按钮

#### 1. **缩小按钮** (-)
- **图标**: 🔍- (FaSearchMinus)
- **功能**: 将内容缩小10%
- **限制**: 最小50%
- **状态**: 达到最小值时按钮变灰不可点击

#### 2. **重置按钮** (100%)
- **图标**: 📏 (FaTextHeight) + 百分比
- **功能**: 重置到默认100%缩放
- **显示**: 当前缩放百分比

#### 3. **放大按钮** (+)
- **图标**: 🔍+ (FaSearchPlus)
- **功能**: 将内容放大10%
- **限制**: 最大200%
- **状态**: 达到最大值时按钮变灰不可点击

### 缩放效果
```css
transform: scale(缩放百分比/100)
transform-origin: top left
width: 100/(缩放百分比/100)%
transition: transform 0.2s ease-in-out
```

## 🔧 技术实现

### 状态管理
```javascript
const [contentZoom, setContentZoom] = useState(100);
```

### 缩放函数
```javascript
// 放大内容
const increaseContentZoom = () => {
  setContentZoom(prev => Math.min(prev + 10, 200));
};

// 缩小内容
const decreaseContentZoom = () => {
  setContentZoom(prev => Math.max(prev - 10, 50));
};

// 重置缩放
const resetContentZoom = () => {
  setContentZoom(100);
};
```

### 样式应用
```javascript
style={{
  transform: `scale(${contentZoom / 100})`,
  transformOrigin: 'top left',
  width: `${100 / (contentZoom / 100)}%`,
  transition: 'transform 0.2s ease-in-out'
}}
```

## 🔄 与其他功能的关系

### 1. **页面缩放**
- 内容缩放独立于页面缩放
- 可以组合使用
- 互不干扰

### 2. **主题切换**
- 缩放功能在浅色/深色主题下都可用
- 控制按钮样式自动适应主题
- 缩放效果保持一致

### 3. **响应式布局**
- 缩放不影响页面布局
- 保持响应式设计
- 适配不同屏幕尺寸
