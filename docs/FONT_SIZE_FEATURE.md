# 字体大小调整功能说明

## 📝 功能概述

为笔记查看和预览页面添加了字体大小调整功能，让您可以根据阅读习惯自由调节文字大小。

## ✨ 功能特点

### 5种字体大小
- **小** (sm) - 适合快速浏览
- **默认** (base) - 标准阅读大小
- **中** (lg) - 舒适阅读大小（原默认）
- **大** (xl) - 大字体阅读
- **特大** (2xl) - 超大字体阅读

### 实时调整
- ➖ **减小** - 点击"-"图标
- ➕ **增大** - 点击"+"图标
- 👁️ **即时生效** - 无需刷新页面

### 智能禁用
- 达到最小时，减小按钮禁用
- 达到最大时，增大按钮禁用

## 🎯 使用位置

### 1. 笔记查看页面
```
/notes/view/:id
→ 顶部操作栏
→ 返回按钮和编辑按钮之间
→ [- 📏默认 +]
```

### 2. 笔记编辑器（预览模式）
```
/notes/editor
→ 点击"预览"标签
→ 字体调整按钮显示在顶部操作栏
→ [编辑][预览][添加图表][- 📏默认 +]
```

## 🎨 界面设计

### 控制器外观
```
┌──────────────────┐
│ [-] 📏 默认 [+]  │
└──────────────────┘
```

### 样式
- 灰色背景圆角容器
- 居中显示当前字体大小
- 左右两个调整按钮
- 禁用状态变灰且不可点击

## 💡 使用示例

### 场景1：阅读长文章
```
1. 打开笔记查看页面
2. 点击 [+] 按钮增大字体到"大"
3. 更舒适地阅读长文章
```

### 场景2：快速浏览
```
1. 打开笔记查看页面
2. 点击 [-] 按钮减小字体到"小"
3. 在一屏内查看更多内容
```

### 场景3：预览笔记
```
1. 在编辑器中写笔记
2. 点击"预览"标签
3. 使用字体调整查看不同大小效果
4. 确保内容在各种字体大小下都清晰可读
```

## 🎯 字体大小对比

### 小 (sm)
- 标题：较小
- 正文：紧凑
- 代码：小号
- 适合：快速浏览、概览内容

### 默认 (base)
- 标题：标准
- 正文：正常
- 代码：标准
- 适合：日常阅读

### 中 (lg) - 之前的默认大小
- 标题：中等
- 正文：舒适
- 代码：清晰
- 适合：长时间阅读

### 大 (xl)
- 标题：大
- 正文：宽松
- 代码：大号
- 适合：护眼阅读、演示

### 特大 (2xl)
- 标题：特大
- 正文：超宽松
- 代码：超大
- 适合：视力辅助、投影展示


## 💡 使用技巧

### 阅读不同类型笔记

#### 算法题解
```
建议：中（lg）或默认（base）
原因：需要看清楚代码细节
```

#### 生活随笔
```
建议：大（xl）或特大（2xl）
原因：放松阅读，减少眼睛疲劳
```

#### 技术文章
```
建议：中（lg）
原因：平衡内容密度和可读性
```

### 屏幕大小适配

#### 大屏幕（27英寸+）
```
建议：大（xl）或特大（2xl）
原因：充分利用屏幕空间
```

#### 笔记本（13-15英寸）
```
建议：中（lg）或默认（base）
原因：平衡内容量和可读性
```

#### 平板/小屏幕
```
建议：默认（base）或小（sm）
原因：在有限空间内显示更多内容
```

## 🎊 组合使用

### 字体调整 + 暗色模式
- 在暗色模式下使用大字体
