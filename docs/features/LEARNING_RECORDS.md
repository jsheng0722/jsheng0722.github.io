# 学习记录（工具栏功能）

## 概述

「学习记录」是工具栏下的聚合功能，用于管理**笔记**以及未来的**英语学习**、**编程语言学习**等子模块。笔记从「内容」中独立出来，归入本功能，便于与学习类扩展（如单词本）统一入口。

## 入口与结构

- **导航**：顶部导航 → **学习记录** → 笔记、**单词本**、**学习资料** /（后续）编程语言 等
- **路由**：
  - `/notes`：笔记首页
  - `/notes/editor`：写笔记 / 编辑
  - `/notes/view/:id`：查看笔记
  - **`/vocabulary`**：单词本（已实现）
  - **`/learning-materials`**：学习资料（已实现，学习文件/链接条目管理）
  - （规划）编程学习等路由

## 笔记分类（记录类型）

笔记按「记录类型」划分，便于区分学习与日常：

| 分类       | 说明                     | 备注                         |
|------------|--------------------------|------------------------------|
| 学习笔记   | 系统化学习内容、课程笔记 | 可扩展：英语学习、编程语言等 |
| 日常日记   | 每日记录、生活流水       | 与原「生活」对应，兼容旧数据 |
| 随笔写写   | 随想、杂文、非正式写作   | 与原「随笔」对应，兼容旧数据 |
| 算法       | 算法题解、LeetCode 等    | 保留算法专属字段与可视化     |

- **兼容**：旧笔记中 `category` 为「生活」的视为「日常日记」，「随笔」视为「随笔写写」，筛选与统计时做映射或合并显示。

## 当前已实现

- **笔记**：首页、编辑器、查看页；分类筛选（学习笔记、日常日记、随笔写写、算法）；搜索、标签、Markdown、算法可视化等
- **单词本**（`/vocabulary`）
  - **单词字段**：单词、音标、词性、释义、例句、近义词、备注；生词/熟词标记
  - **音标与读音**：支持手动输入或「自动获取」（Free Dictionary API）；音标小键盘辅助输入；列表与表单均可播放读音（无读音 URL 时点击可拉取并保存，或使用浏览器 TTS 朗读）
  - **词根词缀**：根据当前输入的单词匹配词根/词缀，点击「+」选入该词注释；支持在添加单词时「添加新词根/词缀」并自动选入本词；单独「管理词根词缀」可增删改查词根词缀库
  - **筛选与搜索**：左侧总词数/生词/熟词卡片即筛选；支持按单词、释义、例句、词性、近义词、备注、词根词缀搜索
  - **数据与同步**：优先从 `public/content/vocabulary/words.json`、`rootsAffixes.json` 加载；「同步到项目」可写入所选目录（File System Access API）或下载 JSON 到本地，放入上述路径后 push 即可
- **学习资料**（`/learning-materials`）
  - 存储学习资料条目：标题、类型、文件路径/链接、说明、标签
  - 支持搜索、添加、编辑、删除，以及一键打开链接
  - 数据存于 localStorage
