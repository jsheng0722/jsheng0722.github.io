# 字体大小调整功能说明

## 📝 功能概述

为笔记查看和预览页面添加了字体大小调整功能，让您可以根据阅读习惯自由调节文字大小。

## ✨ 功能特点

### 5种字体大小
- **小** (sm) - 适合快速浏览
- **默认** (base) - 标准阅读大小
- **中** (lg) - 舒适阅读大小（原默认）
- **大** (xl) - 大字体阅读
- **特大** (2xl) - 超大字体阅读

### 实时调整
- ➖ **减小** - 点击"-"图标
- ➕ **增大** - 点击"+"图标
- 👁️ **即时生效** - 无需刷新页面

### 智能禁用
- 达到最小时，减小按钮禁用
- 达到最大时，增大按钮禁用

## 🎯 使用位置

### 1. 笔记查看页面
```
/notes/view/:id
→ 顶部操作栏
→ 返回按钮和编辑按钮之间
→ [- 📏默认 +]
```

### 2. 笔记编辑器（预览模式）
```
/notes/editor
→ 点击"预览"标签
→ 字体调整按钮显示在顶部操作栏
→ [编辑][预览][添加图表][- 📏默认 +]
```

## 💡 使用技巧

### 阅读不同类型笔记

#### 算法题解
建议：中（lg）或默认（base）
原因：需要看清楚代码细节

#### 生活随笔
建议：大（xl）或特大（2xl）
原因：放松阅读，减少眼睛疲劳

#### 技术文章
建议：中（lg）
原因：平衡内容密度和可读性
