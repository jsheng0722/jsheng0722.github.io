# 🎯 浮动工具栏使用指南

## ✨ 新功能：可自定义位置的浮动工具栏

我们为编辑器添加了一个强大的浮动工具栏，让您可以在页面任何位置快速访问常用功能！

## 📍 工具栏位置

### 默认位置
- **右下角** - 固定在页面右下角
- 不随页面滚动消失
- 始终可见，方便随时使用

### 可自定义位置
点击工具栏左上角的 ⚙️ 设置按钮，可以选择：
- **右侧** - 页面右侧中间（默认）
- **左侧** - 页面左侧中间
- **底部** - 页面底部中间

## 🎨 工具栏功能

### 主按钮 (蓝色圆形 +)
点击展开所有工具按钮
再次点击收起工具栏

### 工具按钮（展开后显示）

#### 1. 📊 添加/编辑图表（紫色/蓝色）
```
功能：打开流程图/思维导图编辑器
颜色：
- 蓝色 = 还未添加图表
- 紫色 = 已有图表（带 ✓ 标记）
位置：最靠近主按钮
```

#### 2. </> 插入代码块（灰色）
```
功能：快速插入代码块模板
模板：
```javascript
// 在这里输入代码
console.log("Hello World");
```
```

#### 3. 📊 插入表格（绿色）
```
功能：快速插入表格模板
模板：
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
```

#### 4. ≡ 插入列表（橙色）
```
功能：快速插入列表模板
模板：
- 项目1
- 项目2
- 项目3
```

#### 5. " 插入引用（靛蓝色）
```
功能：快速插入引用块模板
模板：
> 这是一段引用文字
> 可以跨越多行
```

## 🎬 使用方式

### 方法1：浮动工具栏（推荐）
```
1. 在编辑器页面查看右下角
2. 看到蓝色圆形 + 按钮
3. 点击展开工具栏
4. 选择需要的工具
5. 自动插入到内容末尾
```

### 方法2：顶部操作栏
```
1. 查看页面顶部
2. [编辑] [预览] [📊 添加图表]
3. 点击相应按钮
```

### 方法3：手动输入
```
直接在内容区域输入 Markdown 语法
```

## 🎯 使用场景

### 场景1：编写长文章
```
滚动到页面底部写内容
→ 需要添加图表
→ 不用滚动回顶部
→ 点击浮动按钮即可
```

### 场景2：快速插入元素
```
写作中需要插入代码块
→ 点击代码按钮
→ 自动插入模板
→ 直接修改内容
```

### 场景3：流程图绘制
```
写完文字内容
→ 点击图表按钮
→ 打开流程图编辑器
→ 设计图表
→ 保存嵌入
```

## 🎊 优势对比

### 浮动工具栏 vs 顶部按钮

#### 浮动工具栏优势：
- ✅ 始终可见（不随滚动消失）
- ✅ 位置可自定义
- ✅ 多功能集成
- ✅ 节省顶部空间
- ✅ 动画效果更好

#### 顶部按钮优势：
- ✅ 更显眼
- ✅ 传统位置
- ✅ 不遮挡内容

#### 建议：
- **新手**：用顶部按钮（更容易发现）
- **熟练用户**：用浮动工具栏（更高效）
- **长文章**：用浮动工具栏（不用滚动）
