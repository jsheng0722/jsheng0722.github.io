
## 📝 歌词文件格式

生成的歌词文件是LRC格式：

```lrc
[ti:歌曲标题]
[ar:艺术家]
[al:专辑]
[by:自动生成]
[00:00.00]歌曲标题 - 艺术家
[00:05.00]
[00:10.00]第一句歌词
[00:15.00]第二句歌词
[00:20.00]第三句歌词
```

**可以手动编辑歌词文件来完善内容！**

## 🔧 高级功能

### 智能文件名解析
系统会自动从文件名提取信息：
- `艺术家 - 歌曲名.mp3` → 自动识别艺术家和歌曲名
- `歌曲名.mp3` → 歌曲名作为标题，艺术家设为"未知"

### 在线歌词搜索
- 自动搜索在线歌词库
- 如果找到歌词，自动转换为LRC格式
- 如果搜索失败，生成编辑模板

### 自动文件组织
- 每个歌曲创建独立文件夹
- 自动移动音频文件到对应文件夹
- 生成所有必要的支持文件
