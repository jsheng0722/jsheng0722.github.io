# 📝 笔记功能指南

## 📖 缩放功能

### 页面缩放（浏览器原生）

笔记页面已改用浏览器原生的缩放功能，更加方便和统一。

#### 使用方法

**键盘快捷键**：
- **放大**：`Ctrl + +` (Windows/Linux) 或 `Cmd + +` (Mac)
- **缩小**：`Ctrl + -` (Windows/Linux) 或 `Cmd + -` (Mac)
- **重置**：`Ctrl + 0` (Windows/Linux) 或 `Cmd + 0` (Mac)

**鼠标滚轮**：
- **放大/缩小**：`Ctrl + 滚轮向上/向下` (Windows/Linux) 或 `Cmd + 滚轮向上/向下` (Mac)

**浏览器菜单**：
- 在浏览器菜单中选择"视图" → "放大" 或 "缩小"

#### 优势

- ✅ **统一体验**：所有网页都使用相同的缩放方式
- ✅ **更精确**：支持任意缩放比例（25% - 500%）
- ✅ **更流畅**：浏览器原生优化，性能更好
- ✅ **记忆功能**：浏览器会记住每个网站的缩放设置
- ✅ **快捷键**：标准的键盘快捷键，用户更熟悉

### 内容缩放（独立控制）

笔记页面支持**内容缩放**功能，允许用户独立调整笔记内容的大小，而不影响整个页面的布局。

#### 功能特点

- **缩放范围**: 50% - 200%
- **缩放步长**: 每次调整10%
- **默认缩放**: 100%
- **平滑过渡**: 缩放变化有0.2秒的过渡动画

#### 功能位置

1. **笔记查看页面** (`/notes/view/:id`)
   - 位置: 操作栏右侧，编辑和删除按钮之前
   - 显示: 缩放控制按钮组

2. **笔记编辑页面** (`/notes/editor`)
   - 位置: 仅在预览模式下显示
   - 显示: 缩放控制按钮组

#### 使用方法

**缩放控制按钮**：

1. **缩小按钮** (-)
   - 将内容缩小10%
   - 最小50%，达到最小值时按钮变灰不可点击

2. **重置按钮** (100%)
   - 显示当前缩放百分比
   - 点击重置到默认100%缩放

3. **放大按钮** (+)
   - 将内容放大10%
   - 最大200%，达到最大值时按钮变灰不可点击

#### 应用范围

内容缩放会影响：
- ✅ 笔记标题和正文
- ✅ Markdown 渲染的内容
- ✅ 代码块
- ✅ 流程图和思维导图
- ✅ 所有笔记内容元素

不会影响：
- ❌ 页面导航栏
- ❌ 操作按钮
- ❌ 侧边栏
- ❌ 其他页面元素

## 📐 列表布局

- **卡片视图**：多列卡片，含分类、标题、摘要、作者/日期、标签、难度等。
- **紧凑视图**：单行列表，左侧笔记名+分类，右侧作者与日期（窗口变窄时可换行）。
- 布局切换按钮在列表页操作栏（搜索框与「写笔记」之间），偏好持久化到 `localStorage`。

## 🗑️ 删除与返回

- **删除**：在笔记详情页点击「删除」→ 输入完整标题确认 → 从 userNotes 移除并写入 notesDeletedIds，列表（含 JSON 与 user笔记）会过滤已删除 id。
- **返回笔记**：从可视化页点击「返回笔记」时，优先用 state，无则用 sessionStorage；详情页无 state 时按 URL id 从 userNotes 加载。
