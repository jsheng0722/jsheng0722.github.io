# 音乐播放器功能说明

## 功能概述

本项目实现了一个静态音乐播放器系统，专为静态网站设计，包括：

### 1. 固定侧边栏音乐播放器
- **位置**: 固定在页面右侧，垂直居中
- **功能**: 
  - 播放/暂停控制
  - 进度条拖拽
  - 音量控制
  - 最小化/展开
  - 快速访问音乐库
- **特点**: 不再可拖拽，但会跟随页面滚动保持在固定位置

### 2. 音乐库页面 (`/music`)
- **完整功能**:
  - 音乐列表展示（网格/列表视图）
  - 搜索和筛选功能
  - 按标题、艺术家、年份、类型排序
  - 音乐下载功能
  - 喜欢/取消喜欢功能
  - 播放控制
- **静态设计**: 无需后端支持，所有音乐文件静态存储

### 3. 音乐数据管理
- **数据源**: `public/music/musicList.json`
- **支持格式**: MP3, WAV, OGG, AAC 等音频格式
- **文件存储**: 音乐文件直接存储在 `public/music/` 文件夹
- **数据结构**:
  ```json
  {
    "id": 1,
    "title": "歌曲标题",
    "artist": "艺术家",
    "album": "专辑",
    "duration": "时长",
    "genre": "类型",
    "file": "/music/文件名.mp3",
    "cover": "封面图片链接",
    "year": "年份",
    "liked": false
  }
  ```

### 4. 响应式设计
- 支持桌面端和移动端
- 暗黑模式兼容
- 国际化支持（中英文）

## 使用说明

### 访问音乐播放器
1. 在任何页面的右侧可以看到固定侧边栏音乐播放器
2. 点击播放器上的"音乐"链接或导航栏的"音乐"按钮进入音乐库

### 添加音乐文件
1. **放置文件**: 将MP3文件复制到 `public/music/` 文件夹
2. **更新数据**: 编辑 `public/music/musicList.json` 文件添加音乐信息
3. **刷新页面**: 保存后刷新浏览器页面即可看到新音乐

### 音乐操作
1. **播放音乐**: 点击音乐卡片或列表项
2. **下载音乐**: 点击音乐卡片上的下载按钮
3. **搜索音乐**: 使用搜索框输入关键词
4. **筛选音乐**: 使用分类下拉菜单筛选
5. **排序音乐**: 使用排序下拉菜单排序
6. **收藏音乐**: 点击心形图标收藏喜欢的音乐

### 播放控制
- 播放/暂停: 点击播放按钮
- 进度控制: 拖拽进度条
- 音量控制: 拖拽音量条或点击静音按钮
- 最小化: 点击播放器右上角的"×"按钮

## 注意事项

1. **静态网站**: 这是专为静态网站设计的音乐播放器，无需后端支持
2. **文件管理**: 音乐文件需要手动放置到 `public/music/` 文件夹
3. **数据更新**: 添加音乐后需要手动更新 `musicList.json` 文件
4. **音频文件**: 确保音频文件路径正确且可访问
5. **浏览器兼容**: 某些音频格式可能需要特定浏览器支持
6. **性能优化**: 大量音乐文件时建议压缩音频文件以提高加载速度

## 故障排除

### 常见问题
1. **音乐无法播放**: 检查文件路径是否正确
2. **音乐不显示**: 确认 `musicList.json` 格式正确且文件存在
3. **样式异常**: 检查 Tailwind CSS 是否正确加载
4. **路由错误**: 确认路由配置是否正确

### 调试建议
1. 打开浏览器开发者工具查看控制台错误
2. 检查网络请求是否成功
3. 确认文件路径和权限
4. 验证组件导入是否正确
