# 流程图/思维导图功能 - 快速开始

## 🎨 新功能：可视化构图

类似Figma的拖拽式流程图和思维导图编辑器！

## ⚡ 快速使用

### 5步创建流程图笔记

1. **访问笔记编辑器**
   ```
   /notes → 点击"写笔记"
   ```

2. **填写基本信息**
   ```
   标题: "算法解题流程"
   分类: 算法
   ```

3. **点击"添加图表"按钮**
   ```
   顶部操作栏 → [添加图表]
   ```

4. **创建流程图**
   ```
   • 点击"添加节点" - 创建新节点
   • 拖动节点 - 调整位置
   • 双击节点 - 编辑文字
   • 拖动节点边缘圆点 - 连接到其他节点
   • 点击"保存" - 完成
   ```

5. **保存笔记**
   ```
   流程图会嵌入到笔记中
   查看笔记时可以看到可交互的流程图
   ```

## 🎯 两种模式

### 流程图（蓝色，矩形）
适合：
- ✅ 算法步骤
- ✅ 工作流程
- ✅ 决策树
- ✅ 项目流程

示例：
```
[开始] → [输入数据] → [处理] → [输出结果] → [结束]
```

### 思维导图（紫色，圆形）
适合：
- ✅ 知识整理
- ✅ 头脑风暴
- ✅ 概念图
- ✅ 学习笔记

示例：
```
       [React]
      /   |   \
  [Hooks][组件][Router]
```

## 🖱️ 基本操作

### 节点操作
- **添加**: 点击"添加节点"按钮
- **移动**: 拖动节点
- **编辑**: 双击节点文字
- **删除**: 选中后按Delete键

### 连线操作
- **创建**: 拖动节点边缘圆点到另一个节点
- **删除**: 选中连线后按Delete键

### 画布操作
- **缩放**: Ctrl + 鼠标滚轮
- **平移**: 拖拽空白区域
- **适应**: 点击控制器的适应按钮
