# 图形编辑器使用指南

## 概述

全新的图形编辑器允许你像使用绘图工具一样，直接拖拽和创建各种基本图形，而不是预设的思维导图或流程图模板。

## 主要功能

### 1. 添加图形

点击工具栏中的图形按钮即可添加：

- **矩形** 🟦 - 适合添加流程步骤、文本框
- **圆形** ⚪ - 适合表示开始/结束节点
- **菱形** 🔷 - 适合表示决策点
- **三角形** 🔺 - 适合表示方向指示
- **星形** ⭐ - 适合标记重点
- **文本** 📝 - 添加纯文本标签

### 2. 颜色选择

提供7种预设颜色：
- 蓝色（默认）
- 紫色
- 绿色
- 红色
- 黄色
- 粉色
- 灰色

点击颜色选择器可以切换形状的颜色。选择颜色后添加的新形状将使用该颜色。

### 3. 连接线条

1. 将鼠标悬停在形状上，会出现连接点（小圆点）
2. 从一个形状的连接点拖拽到另一个形状
3. 释放鼠标即可创建带箭头的连接线
4. 连接线颜色与当前选择的颜色一致

### 4. 编辑操作

- **移动形状**：直接拖拽形状到任意位置
- **编辑文字**：双击形状可以修改其中的文字
- **删除元素**：选中形状或连接线，按Delete键删除
- **缩放画布**：使用鼠标滚轮缩放
- **平移画布**：拖拽空白区域平移视图

### 5. 保存图形

点击右上角的"保存"按钮，图形数据会保存到笔记中。

## 快捷操作

- `Delete` - 删除选中的元素
- `鼠标滚轮` - 缩放画布
- `空格 + 拖拽` - 平移画布
- `双击` - 编辑文字

## 使用场景

- 流程图绘制
- 架构图设计
- 概念图表达
- 思维导图
- UML图示
- 自由绘图

## 与之前版本的区别

**之前**：预设的流程图/思维导图模式，只能添加固定样式的节点

**现在**：自由的图形编辑器，可以：
- 选择不同的基本图形（矩形、圆形、菱形等）
- 自定义颜色
- 自由组合各种形状
- 更像传统的绘图工具（如draw.io、Excalidraw）

## 技术实现

基于 ReactFlow 库，通过自定义节点样式和工具栏来实现类似绘图工具的体验。每个形状都是一个 ReactFlow 节点，连接线是 ReactFlow 的边。
