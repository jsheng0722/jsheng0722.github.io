# 流程图/思维导图编辑器完整指南

## 🎨 功能概述

为笔记系统添加了类似 Figma 的可视化构图功能，支持流程图和思维导图两种模式。

## ✨ 核心功能

### 1. 流程图模式
- 📊 **样式**: 矩形节点，蓝色主题
- 🔗 **连线**: 箭头连接，表示流程方向
- 🎯 **用途**: 工作流、算法步骤、项目流程

### 2. 思维导图模式
- 🎯 **样式**: 圆形节点，紫色主题
- 🌳 **结构**: 树状结构，表示层级关系
- 💡 **用途**: 知识整理、头脑风暴、概念图

## ⚡ 快速使用

### 5步创建流程图笔记

1. **访问笔记编辑器**
   ```
   /notes → 点击"写笔记"
   ```

2. **填写基本信息**
   ```
   标题: "算法解题流程"
   分类: 算法
   ```

3. **点击"添加图表"按钮**
   ```
   顶部操作栏 → [添加图表]
   ```

4. **创建流程图**
   ```
   • 点击"添加节点" - 创建新节点
   • 拖动节点 - 调整位置
   • 双击节点 - 编辑文字
   • 拖动节点边缘圆点 - 连接到其他节点
   • 点击"保存" - 完成
   ```

5. **保存笔记**
   ```
   流程图会嵌入到笔记中
   查看笔记时可以看到可交互的流程图
   ```

## 🖱️ 基本操作

### 节点操作
- **添加**: 点击"添加节点"按钮
- **移动**: 拖动节点
- **编辑**: 双击节点文字
- **删除**: 选中后按 Delete 键

### 连线操作
- **创建**: 拖动节点边缘圆点到另一个节点
- **删除**: 选中连线后按 Delete 键

### 画布操作
- **缩放**: Ctrl + 鼠标滚轮
- **平移**: 拖拽空白区域
- **适应**: 点击控制器的适应按钮
