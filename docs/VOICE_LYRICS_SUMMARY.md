## 🎯 使用方法

### 在音乐播放器中使用

1. **进入音乐页面**
   ```
   访问: http://localhost:3000/music
   ```

2. **选择音乐并播放**
   ```
   点击任意音乐卡片播放
   ```

3. **启动语音识别**
   ```
   在播放控制栏点击麦克风图标 🎤
   ```

4. **开始识别**
   ```
   点击"开始识别"按钮
   允许浏览器使用麦克风权限
   ```

5. **保存结果**
   ```
   识别完成后点击"保存歌词"
   下载LRC格式的歌词文件
   ```

### 独立演示页面

```
打开: VOICE_LYRICS_DEMO.html
在浏览器中直接测试语音识别功能
```
