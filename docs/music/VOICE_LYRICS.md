# 🎤 语音识别歌词生成指南

## ✨ 功能概述

基于 **Web Speech API** 的语音识别歌词生成系统，可以在播放音乐的同时识别歌词，自动生成 LRC 格式的歌词文件。

## 🚀 核心功能

### 1. 智能语音识别
- 🎤 使用 Web Speech API 识别音频中的歌词
- ⏱️ 精确记录每句歌词的时间戳
- 📝 实时生成 LRC 格式歌词文件
- 🎯 支持中文歌词识别

### 2. 无缝集成
- 🎵 集成到现有音乐播放器
- 🔘 一键启动语音识别
- 💾 自动保存识别结果
- 📊 显示识别统计信息

### 3. 用户友好
- 🖱️ 简单的点击操作
- 📱 响应式界面设计
- 🔄 实时进度显示
- ⚠️ 智能错误处理

## 📍 如何使用

### 1. 打开语音识别功能

1. **进入音乐页面**：访问 `/music` 页面
2. **选择音乐**：点击播放任意一首音乐
3. **点击麦克风按钮**：在播放控制栏中找到麦克风图标 🎤
4. **开始识别**：点击"开始识别"按钮

### 2. 识别过程

系统会：
- 🎵 自动播放选中的音乐
- 🎤 使用麦克风识别音频中的歌词
- 📝 实时显示识别结果
- ⏱️ 记录每句歌词的精确时间戳

### 3. 保存结果

识别完成后：
- 📄 自动生成 LRC 格式的歌词文件
- 💾 可以保存到本地或上传到服务器
- 📊 显示识别统计（总时长、识别句数等）

## ⚙️ 技术实现

### Web Speech API
- 使用浏览器原生的语音识别功能
- 支持实时识别和批量识别
- 自动处理识别错误和中断

### LRC 格式
- 标准歌词文件格式
- 包含时间戳和歌词内容
- 兼容主流音乐播放器

## 🔧 注意事项

1. **浏览器支持**：需要支持 Web Speech API 的浏览器（Chrome、Edge 等）
2. **麦克风权限**：首次使用需要授权麦克风权限
3. **网络环境**：语音识别需要网络连接
4. **识别准确度**：取决于音频质量和环境噪音
