## 📝 **使用指南**

### **导入方式**
```jsx
// 统一导入（推荐）
import { Button, Card, Input, Badge, EmptyState } from '../../components/UI';

// 单独导入
import Button from '../../components/UI/Button/Button';
```

### **使用示例**
```jsx
// 按钮
<Button variant="primary" icon={<FaPlus />}>添加</Button>

// 卡片
<Card hover clickable onClick={handleClick}>内容</Card>

// 输入框
<Input label="用户名" icon={<FaUser />} required />

// 徽章
<Badge variant="success" size="small">成功</Badge>

// 空状态
<EmptyState title="暂无数据" action={<Button>添加</Button>} />
```
