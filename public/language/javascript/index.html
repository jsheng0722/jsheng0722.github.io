JavaScript 基础速览

JavaScript 是现代 Web 开发中最重要的脚本语言之一，下列内容演示了变量、函数参数、条件判断与循环等核心概念。

变量（Variables）

使用 let 和 const 声明变量，推荐优先使用块级作用域。

const siteName = 'Tech Docs';
let visitCount = 0;

visitCount += 1;
console.log(`欢迎来到 ${siteName}，第 ${visitCount} 次访问。`);

函数与参数（Functions & Parameters）

函数是复用逻辑的基本单位，可以为参数提供默认值，也可以使用对象解构来传入配置。

function greet(name = '访客', { language = 'zh-CN' } = {}) {
  const message = language === 'en-US' ? `Hello, ${name}!` : `你好，${name}!`;
  return message;
}

console.log(greet('Jihui'));
console.log(greet('Amy', { language: 'en-US' }));

条件判断（Control Flow）

function classifyScore(score) {
  if (score >= 90) return '优秀';
  if (score >= 75) return '良好';
  if (score >= 60) return '及格';
  return '需要加强';
}

console.log(classifyScore(82));

循环与迭代（Loops & Iteration）

除了传统循环语句，还可以使用数组迭代方法处理集合。

const frameworks = ['React', 'Vue', 'Svelte'];

for (let i = 0; i < frameworks.length; i += 1) {
  console.log(`[for] ${frameworks[i]}`);
}

frameworks.forEach(item => console.log(`[forEach] ${item}`));

const uppercased = frameworks.map(item => item.toUpperCase());
console.log(uppercased);
