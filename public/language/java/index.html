Java 是强类型、面向对象的编程语言，以下示例涵盖变量、方法参数、条件判断与循环等入门要点。

方法与参数（Methods & Parameters）

public class Greeter {
    public static String greet(String name, String language) {
        if ("en".equals(language)) {
            return "Hello, " + name + "!";
        }
        return "你好，" + name + "!";
    }

    public static void main(String[] args) {
        System.out.println(greet("Jihui", "zh"));
        System.out.println(greet("Amy", "en"));
    }
}

条件判断（Control Flow）

public static String classifyScore(int score) {
    if (score >= 90) {
        return "优秀";
    } else if (score >= 75) {
        return "良好";
    } else if (score >= 60) {
        return "及格";
    }
    return "需要加强";
}

循环（Loops）

List<String> frameworks = Arrays.asList("Spring", "Micronaut", "Quarkus");

for (int i = 0; i < frameworks.size(); i++) {
    System.out.println("[for] " + frameworks.get(i));
}

for (String framework : frameworks) {
    System.out.println("[enhanced for] " + framework);
}

frameworks.forEach(item -> System.out.println("[lambda] " + item));
