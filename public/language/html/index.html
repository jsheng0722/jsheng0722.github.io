HTML 基础速览

HTML（超文本标记语言）用于描述文档结构，是前端开发的基石，以下示例涵盖常见的结构化写法。

页面结构

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8" />
    <title>技术文档中心</title>
  </head>
  <body>
    <header>...</header>
    <main>页面主体内容</main>
    <footer>版权信息</footer>
  </body>
</html>

语义化标签（Semantic Elements）

<article>
  <h2>学习 JavaScript</h2>
  <p>记录变量、函数与循环的知识点。</p>
  <section>
    <h3>变量</h3>
    <p>使用 let / const 声明。</p>
  </section>
</article>

表单（Forms）

<form action="/subscribe" method="post">
  <label>电子邮箱<
    <input type="email" name="email" required />
  </label>
  <button type="submit">订阅</button>
</form>

表格（Tables）

<table>
  <thead>
    <tr>
      <th>语言</th>
      <th>特点</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>HTML</td>
      <td>定义结构与语义</td>
    </tr>
    <tr>
      <td>CSS</td>
      <td>控制样式与布局</td>
    </tr>
  </tbody>
</table>
