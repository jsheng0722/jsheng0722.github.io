Python Basics

Introduction

Python is an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Variables and Data Types

In Python, variables can be declared and values assigned to it like this:

example_var = 5

Control Structures

Python uses standard control structures such as if, for, and while. For example:

if example_var > 1: print("Greater than 1")

Functions

Functions in Python are defined using the def keyword:

def my_function(): print("Hello, World!")

For more information, visit the official Python documentation.
