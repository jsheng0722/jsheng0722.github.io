# 📝 博客文章示例

这是一个演示如何在博客页面存储文章的示例。

## 📁 博客文件存储位置

```
public/files/documents/markdown/
└── blog/           # 博客文章
```

## ✅ 支持的文件类型

| 类型 | 扩展名 | 说明 |
|------|--------|------|
| Markdown | .md | 博客文章内容 |
| 图片 | .jpg, .png | 文章配图 |

## 📌 文章格式

### 文章结构

```markdown
---
title: "文章标题"
date: "2024-01-15"
category: "技术"
tags: ["React", "前端"]
---

# 文章标题

文章简介...

## 章节一

内容...

## 章节二

内容...
```

### 内容示例

这是一篇关于前端开发的博客文章示例。在这里您可以分享技术心得、项目经验或学习笔记。

**技术要点**:
- 使用 Markdown 编写
- 支持代码高亮
- 可以嵌入图片

```javascript
// 代码示例
const blog = {
  title: '我的第一篇博客',
  author: '作者',
  date: new Date()
};
```

---

**提示**: 博客文章应使用 `.md` 扩展名，存储在 `/files/documents/markdown/blog/` 目录下。
