# AlgorithmVisualizer 组件说明

## 📖 当前状态

**重要更新**：代码驱动功能已移除，现在使用工具驱动系统。

### 变更说明

- ❌ **已移除**：代码执行、函数检测、参数提取等代码驱动功能
- ✅ **已改为**：使用 `VisualizationToolkit` 工具驱动系统
- ✅ **保持兼容**：接口保持不变，`CodeBlock.js` 无需修改

## 🎯 使用方式

### 在笔记中使用

```jsx
<AlgorithmVisualizer 
  code={codeString}  // 代码（当前仅用于显示，未来将用于自动生成工具）
  language="javascript"  // 语言类型
/>
```

### 功能说明

1. **工具驱动模式**
   - 从工具栏添加工具（列表、循环、变量等）
   - 手动配置工具参数
   - 使用步骤控制运行可视化

2. **代码显示**
   - 如果提供了 `code` 参数，会在顶部显示代码信息
   - 当前代码仅用于显示，不会自动解析

3. **未来计划**
   - 代码自动解析功能正在开发中
   - 未来将支持从代码自动生成工具配置

## 🔗 相关组件

- `src/components/Visualizations/VisualizationToolkit.js` - 工具驱动系统核心
- `src/components/CodeBlock.js` - 代码块组件（使用 AlgorithmVisualizer）

## 📝 迁移说明

### 之前（代码驱动）
- 自动检测函数
- 自动提取参数
- 执行代码生成步骤
- 自动选择可视化器

### 现在（工具驱动）
- 手动添加工具
- 手动配置参数
- 手动生成步骤
- 使用统一的工具系统

## 🚀 未来计划

1. **代码解析器**
   - 从代码中识别数据结构
   - 自动创建对应的工具
   - 自动配置工具参数

2. **代码执行集成**
   - 执行代码并跟踪状态
   - 将执行结果映射到工具状态
   - 实现代码驱动和工具驱动的桥接
